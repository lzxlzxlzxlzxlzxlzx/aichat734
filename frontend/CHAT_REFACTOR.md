# 聊天界面重构说明

## 改动概述

### 1. 路由简化
**之前：**
- `/chat` - 聊天首页（会话列表）
- `/chat/session/:sessionId` - 聊天会话页面

**现在：**
- `/chat` - 新对话（空状态）
- `/chat/:sessionId` - 已有对话

**优势：**
- 统一的界面体验
- 更符合用户直觉
- 减少页面跳转

### 2. 布局优化

#### 左侧边栏（280px 固定宽度）
- 完整高度显示
- 新建对话按钮置顶
- 历史会话列表可滚动
- 悬停显示删除按钮

#### 主聊天区域（自适应宽度）
- 顶部：会话标题 + 模型选择 + 操作按钮
- 中间：消息列表（最大宽度 900px 居中）
- 底部：快捷回复 + 输入框

#### 消息气泡
- **用户消息**：右对齐，蓝色渐变背景，右侧圆形指示器
- **助手消息**：左对齐，白色背景，左侧头像图标
- 悬停显示操作按钮（Prompt / 复制 / 回溯）

### 3. 视觉改进

#### 配色方案
- 主色调：温暖的米色/奶油色（`#faf8f5`, `#f5f0ea`）
- 强调色：橙棕色渐变（`#bf6c31` → `#b8653a`）
- 用户消息：蓝色渐变（`#3b82f6` → `#2563eb`）

#### 交互细节
- 平滑的动画过渡（150-180ms）
- 悬停效果（轻微上移 + 阴影）
- 消息进入动画（淡入 + 上滑）
- 毛玻璃效果（backdrop-filter）

### 4. 功能完整性

保留所有原有功能：
- ✅ 创建/删除会话
- ✅ 重命名会话
- ✅ 切换模型
- ✅ 发送消息
- ✅ 回溯消息
- ✅ 复制消息
- ✅ 快捷回复
- ✅ Prompt Inspector
- ✅ 自动滚动到底部
- ✅ Enter 发送 / Shift+Enter 换行

### 5. 响应式设计

- 左侧边栏固定 280px
- 主聊天区域自适应
- 消息最大宽度 900px 居中
- Prompt 抽屉宽度 480px（最大 90vw）

## 使用说明

### 启动新对话
1. 访问 `/chat`
2. 点击左侧"新建对话"按钮
3. 自动跳转到新会话

### 继续历史对话
1. 访问 `/chat`
2. 点击左侧历史会话列表中的任意会话
3. 或直接访问 `/chat/:sessionId`

### 发送消息
- 输入框输入内容
- 按 Enter 发送
- 按 Shift+Enter 换行

### 查看 Prompt
- 点击消息下方的"Prompt"按钮
- 或点击顶部工具栏的 Prompt 图标（查看最新消息）
- 右侧抽屉展示完整 Prompt 信息
